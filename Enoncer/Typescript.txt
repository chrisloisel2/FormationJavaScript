    Introduction aux Promesses et Requêtes HTTP
        Théorie sur les promesses en JavaScript/TypeScript.
        Introduction aux requêtes HTTP avec fetch ou axios.

    Lecture de Données (Read)
        Objectif: Utiliser une requête GET pour récupérer la liste des utilisateurs.
        TP: Écrire une fonction qui utilise fetch pour obtenir la liste des utilisateurs depuis http://82.165.127.44/api/users.

    Création d'un Utilisateur (Create)
        Objectif: Envoyer une requête POST pour ajouter un nouvel utilisateur.
        TP: Créer une fonction pour ajouter un nouvel utilisateur en utilisant fetch avec une requête POST vers http://82.165.127.44/api/users.

    Mise à jour d'un Utilisateur (Update)
        Objectif: Envoyer une requête PUT pour mettre à jour les informations d'un utilisateur.
        TP: Écrire une fonction pour mettre à jour un utilisateur. Cette fonction devra envoyer une requête PUT à http://82.165.127.44/api/users/:id.

    Suppression d'un Utilisateur (Delete)
        Objectif: Utiliser une requête DELETE pour supprimer un utilisateur.
        TP: Créer une fonction pour supprimer un utilisateur avec une requête DELETE à http://82.165.127.44/api/users/:id.

    Gestion des Erreurs et Promesses
        Objectif: Apprendre à gérer les erreurs dans les promesses.
        TP: Modifier les fonctions précédentes pour ajouter une gestion d'erreur avec try...catch.

    Utilisation Avancée des Promesses
        Objectif: Utiliser Promise.all pour des requêtes simultanées.
        TP: Créer un scénario où plusieurs requêtes sont envoyées simultanément et gérées avec Promise.all.
